This project performs end-to-end sales and customer analytics using Python and interactive visualizations.

The goal is to identify:

High-value customers

Repeat customer behavior

Regional performance differences

Profit margin by category

Impact of discounting on profitability

Sales trends over time

🛠 Tools Used

Python

Pandas

Seaborn

Plotly

Jupyter Notebook

📈 Key Business Insights

High-value customers contribute the majority of revenue.

Repeat customers spend significantly more than new customers.

Certain regions consistently outperform others.

Excessive discounting negatively impacts profit.

Sales show strong year-over-year growth.

📊 Dashboard Features

KPI Metrics (Sales, Profit, Orders, Customers)

Interactive Regional Sales Analysis

Discount vs Profit Impact Analysis

Monthly & Yearly Sales Trends

Profit Margin by Category

🚀 How to Run

Clone the repository

Open the Jupyter Notebook

Install required libraries

Run all cells
